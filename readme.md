# Portafolio
Este sitio web fue construido para mostrar mis habilidades técnicas y habilidades de diseño web.
El mismo fue hecho con react, basado en plantilla de vite.
## Comandos
### Instalación
``` bash
~$ npm install
```
### Desarrollo
``` bash
~$ npm run dev
```
### building
``` bash
~$ npm run build
```
